# 要求仕様書 F001（文豪ずんだもん初期公開版）

## 概要

青空文庫で公開されている芥川龍之介の保護期間満了作品から、発話として確認した`「」`内の台詞を抽出し、VOICEVOXずんだもんの生成済み音声で再生する静的Webアプリを提供する。利用者は作者一覧から「あくたがわずんのすけ」を選び、作品別の台詞と音声ボタンを閲覧・再生できる。GitHub Pagesで無料公開し、外部API、認証、個人情報収集を持たない。

## 対象範囲

### 対象

- サイト名「文豪ずんだもん」のトップ・作者一覧
- 作者「あくたがわずんのすけ」の作者ページ
- 芥川龍之介の日本語原著から確認済み台詞を収録した作品一覧
- 台詞表示、音声再生、停止、出典・クレジット表示
- 青空文庫取得・台詞抽出・レビュー・VOICEVOX音声生成の再現可能な更新処理
- GitHub Pages向けの静的ビルドと安全なデプロイ

### 対象外

- 実行時の青空文庫スクレイピング、外部TTS API、利用者PCのVOICEVOX ENGINE接続
- 利用者による文章入力、音声生成、投稿、ログイン、課金、広告、解析
- すべての括弧文字列の無審査公開
- 青空文庫の保護期間内作品、現代翻訳、芥川が翻訳者である作品
- サイト上での素材PSD、VOICEVOXソフトウェア、音声モデルの再配布

## 品質・運用条件

- 初期GitHubリポジトリはprivateとし、リリース承認後にpublicへ切り替えてPagesを有効化する。
- Action参照は検証済み完全commit SHAへ固定する。
- コンテンツ取得は明示的な更新操作でのみ行い、青空文庫へ過剰なリクエストを送らない。
- 音声生成に失敗した台詞は公開対象から除外し、文字一覧まで巻き戻さない。
- 収益化、企業販促、スポンサー導入、対象作者追加は別フィーチャーとし、権利・容量・運用条件を再評価する。
- 日本国外における権利状態は一律に保証せず、サイトの利用条件に日本法を基準に選定した旨を明記する。
